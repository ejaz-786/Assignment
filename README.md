## üîπ Assignment Coverage (Short Overview)

This project covers the following assignment topics:

1. **Semantic HTML & Accessibility**

   - Semantic elements for better structure
   - Accessible components for screen readers

2. **CSS Layout & Responsive Design**

   - Fixed header, sidebar, and content layout
   - Mobile-first responsive design

3. **JavaScript Event Handling**

   - Event delegation with dynamically added elements

4. **React Component Design**

   - Search bar with debounced API calls
   - Loading indicators during API fetch

5. **State Management**

   - Local vs Global state
   - Global state handled using modern React patterns

6. **Performance Optimization**

   - Preventing unnecessary re-renders using `React.memo`, `useCallback`

7. **API Integration & Error Handling**

   - Loading, error, and success states
   - User-friendly UI for slow networks

8. **Frontend Security**

   - XSS prevention techniques
   - Safe vs unsafe rendering examples

9. **Debugging**
   - Identifying and fixing React state update issues
